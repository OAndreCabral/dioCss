Seletores CSS

Bem-vindo!

No css nós possuimos os seletores por ID

Podemos aplicar as mesma regras CSS para vários seletores diferetes. Para isso, podemos separar os seletores utilizando uma virgula(,)

Assim, não precisamos ficar copiando e colando os mesmo códigos para todos, nós podemos fazer um só bloco de CSS e reutilizando as mesma regras!
  • Seletor por Tipo:Busca elementos por uma tag HTML especifica
  • Seletor por ID (#):Busca elementos através do atributo 'ID'
  • Seletor por Classe (.):Busca elementos através do atributo 'class'
  • Seletor por Universal (*):Seleciona todos os elementos HTML
  • Seletor por Atributo ([atrib]):Seleciona elementos que possuem um atributo especifico em nosso documento HTML e conseguimos também buscar atributos com um valor especifico
